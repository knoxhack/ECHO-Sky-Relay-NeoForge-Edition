# ECHO Sky Relay NeoForge Edition

Sky Relay NeoForge Edition is the Minecraft/NeoForge compatibility lane for
`ECHO: Sky Relay`, Official ECHO Pack #3.

## Role

- Consumes `-neoforge.jar` artifacts from `ECHO-Modules`.
- Uses `echoskyrelayprotocol` as the canonical Sky Relay content source.
- Publishes NeoForge install, update, repair, and rollback manifests for the ECHO Launcher.
- Adapts ECHO IDs into NeoForge runtime behavior without making Minecraft content the design source.

## Status

Implementation foundation only. Keep preview-only until data, generated assets,
recipes, fragment anchoring, storm behavior, save/load, and launcher flows pass.
